# Inventory Wishlist (MERN)

A simple MERN app with user auth (JWT), product CRUD and user wishlist.

## Run locally

### Backend

1. cd backend
2. npm install
3. copy .env.example to .env and set MONGO_URI and JWT_SECRET
4. npm run dev

### Frontend

1. cd frontend
2. npm install
3. set VITE_API_URL in frontend/.env (default `http://localhost:5000/api`)
4. npm run dev

Open frontend dev url (Vite usually at http://localhost:5173)
